Куб суммы

Формула: (a + b)³ = a³ + 3a²b + 3ab² + b³

Доказательство:
(a + b)³ = (a + b)(a + b)(a + b) = 
= (a + b)(a² + 2ab + b²) = 
= a³ + 2a²b + ab² + a²b + 2ab² + b³ = 
= a³ + 3a²b + 3ab² + b³

Примеры:
1) (x + 2)³ = x³ + 3x²·2 + 3x·2² + 2³ = x³ + 6x² + 12x + 8
2) (2a + 3b)³ = (2a)³ + 3(2a)²(3b) + 3(2a)(3b)² + (3b)³ = 
   = 8a³ + 3·4a²·3b + 3·2a·9b² + 27b³ = 
   = 8a³ + 36a²b + 54ab² + 27b³

Применение:
- Разложение выражений
- Упрощение алгебраических выражений
- Решение уравнений
