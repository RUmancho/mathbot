Сумма кубов

Формула: a³ + b³ = (a + b)(a² - ab + b²)

Доказательство:
(a + b)(a² - ab + b²) = a³ - a²b + ab² + a²b - ab² + b³ = a³ + b³

Примеры:
1) x³ + 8 = x³ + 2³ = (x + 2)(x² - 2x + 4)
2) 27a³ + 64b³ = (3a)³ + (4b)³ = (3a + 4b)(9a² - 12ab + 16b²)
3) 8x³ + 1 = (2x)³ + 1³ = (2x + 1)(4x² - 2x + 1)

Применение:
- Разложение на множители
- Упрощение дробей
- Решение уравнений
- Доказательство тождеств
