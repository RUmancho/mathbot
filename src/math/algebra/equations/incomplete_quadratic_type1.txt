Решение

Для решения уравнения вида
ax² + bx = 0 необходимо
вынести x за скобку

x(ax + b) = 0
далее приравнять выражение
в скобках к нулю

ax + b = 0

А далее решать получившееся
линейное уравнение
Причём уравнение имеет 2 корня,
один из которых всегда 0

Пример

2x² - 32x = 0 
x(2x - 32) = 0
2x - 32 = 0
2x = 32
x = 32 : 2
x = 16

x₁ = 0
x₂ = 16
