Решение

Для решения уравнения вида
ax² + c = 0
необходимо перенести коэфициент
"c" после знака "=", далее 
нужно разделить коэфициент
"c" на коэфициент с неизвестной "x"
далее нужно вытащить из-под корня
коэфициент "с". уравнение имеет 2
корня, один со знаком "+", другой
со знаком "-"

пример

5x² - 125 = 0
5x² = 125
x² = 125 : 5
x² = 25
x = √25

x₁ = 5
x₂ = -5
