Вид
ax² + bx + c = 0

Решение по дискриминанту

Для нахождения корней, нужно
Воспользоваться формулой
D = b² - 4ac

Далее, если D > 0, тогда считаем корни

x₁ = -b + vD : (2a)
x₂ = -b - vD : (2a)

но, если же дескриминант < 0,
то тогда решения уравнения нет,

Пример

x2 - 4x - 5 = 0 
a = 1; b = -4; c = -5

D = -4² - 4 × (-5)
D = 16 + 20 = 36

x₁ = (-(-4) + v36) : 2
x₁ = (4 + 6) : 2 = 5

x₂ = (-(-4) - v36) : 2
x₂ = (4 - 6) : 2 = -1

x₁ = 5
x₂ = -1
