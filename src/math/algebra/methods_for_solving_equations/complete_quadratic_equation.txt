Решение уравнения вида 
(4x + 2)(x + 5) = 0

Это уравнение -- квадратное, ведь 
раскрытие скобок даёт уравнение 
второй степени (4x² + 22x + 10)
Его можно решить двумя способами

1-й способ быстрый: Использование свойства нуля

Если произведение двух множителей равно нулю (a × b = 0), 
то хотя бы один из множителей равен нулю (или a = 0, или b = 0) 
(  по аналогии: a = (4x + 2), b = (x + 5)  -- 2 множителя) 
Значит, запишем что
4x + 2 = 0 или x + 5 = 0

Решаем оба линейных уравнения:

4x + 2 = 0      x + 5 = 0 
4x = -2         x = -5
x = -0,5        

Ответ: x₁ = −0,5; x₂ = −5


2-й способ: Раскрываем скобки

После раскрытия уравнение выглядит так
4x² + 20x + 2x + 10 = 0  (cложим иксы ...)
4x² + 22x + 10 = 0

Далее используем любой метод решения
квадратного уравнения 
(через дескриминант, или по теореме Виета)

Ответ будет таким же: x₁ = −0,5; x₂ = −5
