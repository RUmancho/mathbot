Тригонометрические уравнения

Основные типы уравнений:

1) sin x = a
   Решение: x = arcsin a + 2πn, x = π - arcsin a + 2πn, n ∈ Z

2) cos x = a
   Решение: x = ±arccos a + 2πn, n ∈ Z

3) tg x = a
   Решение: x = arctg a + πn, n ∈ Z

4) ctg x = a
   Решение: x = arcctg a + πn, n ∈ Z

Примеры:
1) sin x = 1/2
   x = π/6 + 2πn, x = 5π/6 + 2πn, n ∈ Z

2) cos x = -1/2
   x = ±2π/3 + 2πn, n ∈ Z

3) tg x = 1
   x = π/4 + πn, n ∈ Z

Методы решения:
- Приведение к простейшим уравнениям
- Разложение на множители
- Замена переменной
- Использование формул приведения
